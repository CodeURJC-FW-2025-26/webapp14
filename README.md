# Project - Part 2

## Execution Instructions

### Prerequisites

Before running the project, make sure the following are installed:

-> Node.js 
-> MongoDB 

1. Download the project
   
git clone <repository-url>
cd webapp14/SellOra

3. Install dependencies

`npm install`

4. Run the application

`npm run watch`

5. Open the website

Open your browser at:

`http://localhost:3000`

### File Description

/src
app.js – Main entry point of the server. Sets up Express, middleware, routes, and connections.
load_data.js – Functions for loading data from JSON or database.
product.js – Handles product-related operations (finding, creating, updating…).

router.js – Defines all Express routes for the application.
store.js – Logic related to database storage or in-memory data handling.

/public
css/ – Stylesheets used across the website.

img/ – All static images used in the web app (product photos, icons, etc.).
(Note: This is where your actual images currently are.)

data/ 
data.json – Example product data.
images/ – Can be used for storing product images loaded from JSON if needed.

/uploads

Folder for storing uploaded product images.

/views

HTML templates rendered by the server:
 - partials/
      header.html – Reusable header section.
      footer.html – Reusable footer section.

SELLORA.html – Homepage.
upload.html – Page for adding a new product.
detail.html – Product detail page.

updated_product.html – Confirmation page after editing a product.
